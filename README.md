ІНСТРУКЦІЯ ІЗ ЗАПУСКУ:

1. База Даних (PostgreSQL):
   - У репозиторії знаходиться файл "script.sql".
   - Створіть порожню базу даних (наприклад, "CinemaDB").
   - Запустіть цей скрипт у pgAdmin (Query Tool), щоб створити таблиці та наповнити їх даними.

2. Налаштування:
   - Відкрийте файл "appsettings.json" у проєкті.
   - Змініть рядок "DefaultConnection" на актуальний для вашого ПК.

3. Запуск:
   - Відкрийте рішення "Cinema.sln" у Visual Studio.
   - Запустіть проєкт (профіль https).
   - Автоматично відкриється Swagger UI для тестування API.
